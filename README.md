## 共用元件整合包

專案結構如下

- commons-base： 共用基礎
- commons-design： 設計模型
- commons-thread： 平行化
- coomon-prooptics： 代碼元件
- hcp： Hitachi Content Platform 雲端平台共用元件
- hcp start： spring boot start 
- telegram： tg組合共用元件
- commons-scheduling： 排程服務
- commons-crypto： 加密
- commons-mybatis： 輔助工具
- commons-textcut： 文字切割工具
- commons-check： 資料驗證共用元件
